# 运维工程师的简明Python教程

本书来源于《A Byte of Python》，它是一本由 **Swaroop C H** 编写，旨在于介绍如何使用 Python 语言进行编程的自由图书。它以教材与指南的形式为入门者介绍 Python 语言。如果你对电脑知识的了解仅限于如何保存文本文件的话，那这本书就是为你准备的。

2005 年，**沈洁元**将本书的 1.20 版引进中国，并完成了全本翻译，将本书的译名定为《简明 Python 教程》。2017年，**漠伦**基于原书 4.0 版重新翻译，制作了本版全新译本，并沿用同样的译名。

2018年，**赵班长**在基于**漠伦**的全新翻译版本基础上进行修改，主要是为了满足将本书的一些案例修改为运维工程师相匹配的内容，帮助国内的运维工程师完成SHELL能力到Python能力的转变。

本版本定义为 v.10 版。最后修订于 2018 年 4 月 16 日，将单独进行演进。

## 有关 Python 3

本书将指导你如何使用 Python 3。同时，本书也会以指南的形式告诉你应如何适应虽老些但使用更加普遍的 Python 2。
